# PrediData Insights

PrediData Insights is a machine-learning powered sales forecasting and product segmentation platform designed to help businesses make data-driven decisions. It analyzes historical sales patterns, predicts future demand, and categorizes products into economic, mid-range, and premium tiers to support strategic planning and market segmentation.
