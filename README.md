# PrediData Insights

PrediData Insights is a machine-learning powered sales forecasting and product segmentation platform built to help businesses make informed, data-driven decisions. It analyzes historical sales data, forecasts future trends, and classifies products into economic, mid-range, and premium tiers to support pricing and market strategy.

---

## 🚀 Features

### 📊 Sales Forecasting  
- Achieved **92.3% prediction accuracy** using advanced neural networks.  
- Included **dropout layers**, **batch normalization**, and the **AdamW optimizer** for improved performance and stability.

### 🎯 Product Segmentation  
- Automatically groups products into:
  - Economic  
  - Mid-range  
  - Premium  
- Helps businesses understand market tiers and optimize pricing strategies.

### 🌐 User-Friendly Web Interface  
- Backend crafted with **Flask** for seamless model integration.  
- Frontend designed with **HTML**, **CSS**, and **JavaScript** for an intuitive experience.  
- Provides easy access to predictions and insights.

### 📈 Visual Analytics  
The system generates useful analytical visualizations including:  
- Scatter plots  
- Bar charts  
- Distribution graphs  

These visuals simplify complex data insights for easier understanding.
